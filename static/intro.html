Welcome to the Visual Analytics Study

This study investigates how humans perceive and interpret 2D scatter plots derived from high-dimensional data using various dimensionality reduction methods. Your participation will help researchers understand which visual representations are most effective for identifying patterns and structures in complex data.

Study Procedure

  • The study will take approximately 30–40 minutes in a single session.
  • Accuracy is more important than speed — take your time to answer carefully.
  • The study must be completed in one sitting, but short breaks are allowed (e.g., for restroom use).
  • You can click the Pause button during the study to temporarily hide the screen and resume later.

What is a Scatter Plot?

A scatter plot is a two-dimensional visualization where each point represents one data item. The distance between points reflects their similarity in the original high-dimensional space: similar items appear close together, while dissimilar items appear farther apart. We use the Palmer penguin dataset for all scatterplot in this introduction.

The Penguins dataset, which contains measurements for three penguin species from the Antarctic. Features like body mass and flipper length can overlap between species.

What are Clusters?

Clusters are groups of points that form visually coherent regions in the scatter plot. These clusters often represent similar items in the underlying data. You can recognize them by looking for dense regions that are spatially separated from others. Clusters can vary in:

  • Size: the number of points in the cluster
  • Density: how closely packed the points are within a given area
  • Shape: clusters can be round, elongated, or irregular

Measurements like flipper length and bill length separate penguin species clearly.

Well-separated clusters are easy to identify. This could be a possible selection for the first task in the study.

Ground Truth vs. Visual Clusters

In many datasets, each data point belongs to a known category — for example, in the Penguin dataset, each penguin is identified as a specific species. These known categories are called the ground truth.

However, when looking at a scatter plot, we can only see how points are arranged in two dimensions. Visual clusters are the groups that can be recognized directly from the plot, based solely on the spatial arrangement of points — regardless of their true category.

In this study, all points will be shown in gray without category labels. This means you will need to identify clusters based off their visible structure, without any hints from the original data classes. The examples shown are possible clusters selected from a human perception.

Using the Lasso Tool

  • Click and hold the left mouse button to start drawing a selection shape.
  • Draw around the points you wish to select, then release the mouse button to complete the selection.
  • Click the Reset button to clear your current selection.
  • A point can belong to only one cluster. If you lasso a point again, it will be removed from its previous cluster and assigned to the new one.
  • Not all points have to be in a cluster — leave unclustered points as they are if they don't clearly belong to a group.

What Will You Be Doing?

You will be shown several scatter plots and asked to perform different tasks:

  • Select clearly visible clusters using the lasso tool
  • Identify the cluster containing a highlighted (red) point
  • Select the cluster closest to a reference cluster
  • Choose the most densely packed cluster
  • Rank all projections based on clarity and visual structure

Each task block starts with an example to help you understand the task. The full study takes approximately 30–40 minutes. Your responses will be recorded anonymously.
